# 🚀 Resume Analyser AI

An AI-powered resume analysis tool that evaluates resumes against a job description and provides structured insights, technical skill extraction, and brutally honest feedback using Large Language Models.

Built with **Python, Streamlit, Groq API, and Llama 3.3 70B**.

---

## 📌 Overview

Resume Analyser AI helps candidates understand how their resume performs for a specific job role.

The application extracts important technical skills, analyzes project experience, detects unnecessary buzzwords, and generates an AI-powered hireability score with detailed feedback.

---

## ✨ Features

✅ Upload multiple resumes in PDF format  
✅ Extract resume text automatically  
✅ Analyze resumes against a given job description  
✅ Identify Web Development technologies  
✅ Detect Data Structures & Algorithms experience  
✅ Generate a Hireability Score (1-10)  
✅ Detect overused corporate buzzwords  
✅ Provide brutally honest AI-generated feedback  
✅ Clean and interactive Streamlit dashboard
